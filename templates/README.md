# PDF Templates
This folder hosts the PDFs in their template form that is used by the PDF-service to attach dynamic data.
For simplicity, only the digital variants are shown. Physical variants are inspected internally by Trifork.

---

Dynamically inserted text is enclosed in a pair of dollar signs (`$`) and use the name of their substitution value.
Note the location of dynamic fields is not precise but should give an idea as to where and what kind of data is inserted.

See (the wikki)[https://github.com/trifork/dpk-docs/wiki/Templates]
